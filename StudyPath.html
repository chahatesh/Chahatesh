The Idea
Project Overview

Many students spend hours studying but struggle to see results — not because they lack effort, but because their study methods don't match how they learn. StudyPath is an adaptive learning platform that solves this by combining quizzes, personalized feedback, progress tracking, and structured tools into one unified system.

Instead of a one-size-fits-all approach, StudyPath identifies each user's learning style at the start — visual, auditory, or hands-on — and uses that profile throughout the entire experience. Mistakes become learning moments. Feedback becomes personal. Studying becomes more effective.

Learning Styles
👁️

Visual Learner

Learns through images and diagrams. Gets diagram-based hints and mental visualization prompts when they answer incorrectly.

🎧

Auditory Learner

Learns through listening and verbal explanation. Guided to read through and talk out explanations step by step.

🛠️

Hands-On Learner

Learns through doing and practice. Receives detailed step-by-step reasoning guidance that walks through each problem.

Under the Hood
Tech Stack
🔥
Firebase — The Backbone of StudyPath
Firebase was chosen as the primary backend because it makes content management effortless for moderators. Instead of editing code every time a quiz question needs to change, moderators and admins can add, edit, or remove questions directly from the Firebase console — no redeploy needed. This keeps the question bank fresh and growing without any developer involvement. On top of that, Firebase handles user authentication, real-time data syncing for progress and quiz state, and a flexible database structure that scales as new subjects and features are added.

Development History
Work Timeline
Project Kickoff

Identified the core problem: students study hard but often use methods that don't match how they learn. Defined the three learning styles and began designing the adaptive quiz system around them.

Firebase Setup

Set up Firebase Authentication and Firestore. Designed the database structure to support multiple subjects, quiz questions per subject, and user-level progress tracking — with moderator roles built in from the start.

Quiz Engine

Built the adaptive quiz engine. The key logic: when a user gets a question wrong, the system checks their learning style profile and returns a different type of hint rather than just showing the answer.

Dashboard & Notes

Built the progress dashboard showing quizzes completed and average score. Added the built-in notes system so students can jot down key information without leaving the app.

Learning Path Mode

Implemented the infinite practice mode — an adaptive loop that continuously generates or recycles questions to reinforce concepts through repetition instead of stopping after a single quiz.

In Progress

Currently refining the learning style detection, expanding the subject library, and exploring AI-powered explanation generation for more dynamic and personalized feedback at scale.

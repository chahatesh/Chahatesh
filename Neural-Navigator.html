The Idea
Project Overview

Neural Navigator is a project I built in 2026 while in 8th grade, aimed at making AI concepts more accessible and easier to explore for students, hobbyists, and beginners. The idea came from noticing how difficult it can be to find the right AI tools for specific tasks — information is often scattered across multiple websites, forums, and platforms.

This project organizes AI tools and resources into a structured, searchable database, allowing users to browse categories, find specific tools, and interact with an intuitive interface that guides them to the right resources quickly and efficiently. Neural Navigator solves the problem of scattered AI resources — helping users find tools without wasting time.

Started building the search feature so users can find AI tools quickly. Faced issues with filtering results correctly and had to revise the Firestore database queries to work in real time.

Polished the search UI — added sorting options and redesigned the results layout for readability. Small UX improvements made a noticeable difference in usability.

April 1, 2026

Integrated the search with the live AI database. Fixed a bug where the tool list wasn't updating in real time — turned out to be a Firestore listener issue not refreshing state correctly.

Finalized the Replit deployment and ran thorough end-to-end testing. Made last-minute UI polish, wrote documentation, and confirmed everything works across screen sizes.
